Lesson 1: The Basics of Web and HTML

The Web starts with a user who, through a browser, accesses the internet. The internet is the largest network of computers in the world. Computers that host the files that make up the web are called servers. HTTP is the main protocol of the web.

A Browser interprets and displays web pages that are text documents written in HTML.

HTML, or HyperText Markup Language, documents are the heart of the web. It includes text content, markup, references to other documents and links to other pages. Here's a link to some more information.

HTML vocabulary

Tag
an HTML tag is always enclosed in angeled brackets. These contain the instructions that the browser reads.
Element
an HTML element describes everything between the opening and closing tags
Attribute
an attribute is a property of an HTML element.

Coding is writing text in a language computers can understand, such as HTML. Computers are stupid and can't interpret meaning from text unless unless it is written perfectly. However, computers will never make erroneous interpretations. If a coder can't remember the details it is possible to simply look them up.

Lesson 2: Creating a Structured Document with HTML

Creating structure using HTML is easily done using a "Tree-Like Structure". This refers to the branching out used in HTML. Similar to file paths in a documents folder. Within the text of an HTML document this appears as a series of indented elements.

Structure on a web page can be identified as a series of Boxes. Boxes are an important part of web page structure and design.

Block elements of HTML, as opposed to line elements, plus the tree-like structure of HTML is used to create the boxes that appear on a web page. The further indented a line is in HTML, the deeper nested the corresponding box is on the web page.

Text Editors, such as Sublime Text, make this entire process much easier.

Lesson 3: Adding CSS Style to HTML Structure

CSS or Cascading Style Sheets adds style to HTML structure. The cascading aspect of CSS is also known as inheritance, where previous specifications are applied to the follow on elements.

Avoiding Repetition is an importnat aspect of CSS. Avoiding repition helps to reduce errors and increase consistency.

Selectors target specific HTML elements and apply style to them. One of the most best selectors to use is a class selector. Selectors are instrumental in avoiding repetition. A better explanation, and more information, can be found here.

After a selector there should be a pair of curly braces. Inside the braces there is a declaration which consists of a property and a value. The declaration is a rule that is applied to the selector.

CSS has a lot of functionality, it would be extremely difficult to memorize all CSS attributes. Luckily coders can simply look up what they need from CSS documentation.

There are three methods for adding CSS to HTML. The most practical method is to create a seperate file for your CSS code and link it to the HTML, as is demonstrated here.

Boxes are pretty important with web page development, and CSS can help with the placement and sizing of boxes. Andy Brown has provided us with a handy resource page, which includes all this CSS layout info.
